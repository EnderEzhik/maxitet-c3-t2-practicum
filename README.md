## Идея
Сайт с каталогом модов для minecraft.

### Цель
Сделать сайт с каталогом модов для minecraft с удобными фильтрами по версии игры и категориям.\
Главной особенностью будет автоматическое скачивание всех необходимых зависимостей для мода.

---

### Как запустить проект
1. Клонируйте репозиторий

`git clone https://github.com/EnderEzhik/maxitet-c3-t2-practicum`

2. Установите зависимости

`pip install -r requirements.txt`

3. Перейдите в директорию с кодом

`cd backend`

4. Примените миграции (alembic не умеет создавать саму базу данных, поэтому её нужно создать вручную - EasyMods)

`alembic upgrade head`

5. Запустите API сервер

`uvicorn src.main:app --reload`

6. Откройте статический html файл в директории frontend

### Роли и их права
- Гость
    - Может просматривать каталог модов
    - Может скачивать моды
- Пользователь
    - Всё, что может гость
    - Может оценивать моды
    - Может оставлять комментарии к модам
    - Может отправлять на модерацию собственные моды для добавления их в каталог
    - Может добавлять моды в избранное
    - Может просматривать историю скачанных модов
- Модератор
    - Всё, что может пользователь
    - Может управлять заявками на модерацию модов (принимать/отклонять)
    - Может управлять уже опубликованными модами
    - Может управлять списком версий
    - Может управлять списком категорий
- Администратор
    - Всё, что может модератор
    - Может управлять пользователями
    - Может управлять модераторами (назначать, снимать)

### User story
- Гость
    - Я хочу просматривать каталог модов и использовать фильтры для быстрого поиска
    - Я хочу иметь возможность открыть понравившийся мод в каталоге и увидеть его страницу/карточку
    - Я хочу иметь возможность скачивать моды
- Авторизованный пользователь
    - Всё тоже, что и у гостя
    - Я хочу иметь возможность оставить комментарий к моду
    - Я хочу иметь возможность загрузить свой мод
- Модератор
    - Всё тоже, что и у авторизованного пользователя
    - Я хочу иметь возможность модерировать заявки на публикацию мода и уже опубликованные моды
- Администратор
    - Всё тоже, что и у модератор
    - Я хочу иметь возможность управлять пользователями и модераторами


### Модели данных
#### Mod
- id: uuid
- title: str
- description: str
- rating: float
- author: str
- published_by: uuid
- categories: list[category]
- versions: list[version]
- dependencies: list[mod]

#### Version
- version: str

#### Category
- title: str

#### User
- id: uuid
- username: str
- login: str
- password: str
- is_active: bool
